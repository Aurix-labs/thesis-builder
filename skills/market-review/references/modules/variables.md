> 本模块产出：海外市场收盘、政策新闻影响评级
> 数据输入：`output/<ymd>/variables/data.json` + Agent WebSearch
> 输出文件：同目录 `report.md`，必须以 `<!-- REVIEW_MODULE_START module=variables date=<ymd> -->` 开头

## 强约束

- 新闻部分由 Agent 通过 WebSearch 获取

- 市场复盘不提个股
