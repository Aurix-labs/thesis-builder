# capital · 资金行为监测 · 方法论

> 本模块产出：全市场资金流向、北向资金解读、龙虎榜信号
> 数据输入：`output/<ymd>/capital/data.json`
> 输出文件：同目录 `report.md`，必须以 `<!-- REVIEW_MODULE_START module=capital date=<ymd> -->` 开头

## 数据时效说明

| 数据源 | 盘后可用时间 | 字段 |
|--------|------------|------|
| `stock_market_fund_flow` | **15:30 立即可用** | 主力/超大单/大单/中单/小单净流入 |
| `stock_hsgt_hist_em` | 15:30（参考字段）/ 18:00（净买额） | 上证指数、领涨股（立即可用）；净买卖额（延迟） |
| `stock_sina_lhb_detail_daily` | 约 16:30 后 | 龙虎榜上榜个股、买卖席位 |

## 强约束

- 最低行数：25 行
- 必含 section：全市场资金流向 / 北向资金 / 龙虎榜信号
- 市场复盘不提个股
- 龙虎榜和北向净买额为空时，必须在报告中标注"待盘后更新"而非视为数据异常

## 分析方法

### 全市场资金流向（盘后立即可用）

| 指标 | 数值 |
|------|------|
| 主力净流入 | [F:fund_flow.today.主力净流入-净额] 亿（正值=流入，负值=流出） |
| 主力净占比 | [F:fund_flow.today.主力净流入-净占比]% |
| 超大单净流入 | [F:fund_flow.today.超大单净流入-净额] 亿 |
| 大单净流入 | [F:fund_flow.today.大单净流入-净额] 亿 |
| 中单净流入 | [F:fund_flow.today.中单净流入-净额] 亿 |
| 小单净流入 | [F:fund_flow.today.小单净流入-净额] 亿 |

判断：[I:] 主力净流入/流出方向及力度。主力大幅流出+小单大幅流入 = 散户接盘，偏空。

### 北向资金

| 指标 | 数值 |
|------|------|
| 当日净买卖 | [F:northbound.today_net] 亿（约 18:00 后更新，15:30 通常为 null） |
| 沪股通最新上证收盘 | [F:northbound.recent_10d[-1].上证指数]（参考锚） |
| 连续3日 | [I:]（判断持续流入/流出/转向） |

> 若 today_net 为 null，不要报错。用全市场资金流向 + 上证指数涨跌推断北向大致方向。
