## 一、研究原则（执行约束）

agent 在 Phase 2 执行 Step 0-8 时必须贯彻以下原则。

### 1.1 七条原则

1. **产业驱动非情绪驱动** — 顺产业趋势，不顺 K 线趋势
2. **先验证后结论** — 完成"趋势三要素"再下判断，基本面 + 资金面 + 技术面三重交叉验证
3. **先风控后收益** — 每份报告必须写清止损位与逻辑破坏条件
4. **盈亏比思维** — 量化向上空间 vs 向下空间，三档目标价（短 / 中 / 长）
5. **看大象比看蚂蚁容易** — 优先大市值、信息透明度高的龙头；百亿以上优先，60 亿以下慎重
6. **低位谈逻辑、高位讲情绪** — 位置决定分析重心；高位关注拥挤度和资金面
7. **宁可错过不可做错** — 等业绩拐点确认，识别"真成长"与"伪题材"

### 1.2 三条底线（不可违反）

1. **先事实后推断** — 关键结论必须有数据日期 + 来源
2. **先验证后判断** — 至少"趋势三要素"齐全
3. **先风控后收益** — 失效条件必须显式列出

### 1.3 方法论闭环

> 先市场定价 → 反推产业逻辑 → 回到公司业绩验证 → 交易与风控

产业层面自上而下扫描（宏观 → 产业链 → 细分产品），个股层面按 Step 0-8 执行。

## 二、分析流程（Step 0 → Step 8）

### Step 0：任务锁定

> **inputs:** `meta`, `quote`
> **outputs:** 标的元信息 + 周期/状态/风格预判

| 字段 | 要求 |
|------|------|
| 标的 | 公司名、代码、交易所 |
| 周期 | `短线(1-4周)` / `中线(1-2季)` / `双周期` |
| 数据截止日 | 必须写绝对日期（YYYY-MM-DD） |
| 研究状态 | `首次覆盖` / `跟踪更新` / `事件复盘` |
| 风格预判 | `配置型` / `交易型` / `左侧博弈型` |

### Step 0.5：核心异常分析（v3.1 新增 · 必做）

> **inputs:** `blocks.financial_abstract`, `blocks.fund_flow`, `kline_daily`, anomalies.json
> **outputs:** 每条 CRITICAL/HIGH 异常的成因 + 影响 + 跟踪锚点

**前置：** 必须先读 `anomalies.md`（Phase 1.5 自动产出）。

**强制：** 对每条 **CRITICAL** anomaly 必须有一整段 ≥80 字分析；对每条 **HIGH** 必须列出。

**模板：**

```text
#### A001 · <indicator> <period> <YoY 描述>
- 数值：<value> [F:<blocks_ref>]
- 上年同期：<prior_value> [F:<prior_ref>]
- **可能原因**：<原因 1 / 原因 2 / 原因 3>，需在 Step <X> 深入
- **对投资逻辑的影响**：<拐点是否成立 / 估值是否需要重置>
- **跟踪指标**：<在 Step 8 跟踪锚点中加入对应字段>
```

**输出：** 一整段 ≥80 字分析。verify_facts.py 会检查 CRITICAL 的 indicator 字符串是否在本段出现。

### Step 1：宏观与周期定位

> **inputs:** `meta.industry`, `blocks.news`
> **outputs:** 经济周期定位 + 政策环境 + 核心矛盾一句话

#### 1a. 经济周期映射
- 经济阶段：`复苏前期` / `复苏中期` / `复苏后期` / `震荡`
- 公司所处周期与宏观的映射关系
- 政策阶段：`准备期` / `发力期` / `落地分化期`

**经济周期与板块对应：**

| 周期阶段 | 优势板块 | 逻辑 |
|---------|---------|------|
| 复苏前期 | 资源涨价（有色/稀土/化工） | 上游供给刚性，价格先行 |
| 复苏中期 | 设备/制造放量 | PMI回暖→机械设备订单 |
| 复苏后期 | 消费跟随 | 居民收入改善滞后传导 |

#### 1b. 政策与环境扫描
- 核心政策驱动（具体到政策名称、文件编号）
- 外部环境：汇率、地缘、海外需求、监管变化

#### 1c. 核心矛盾提炼（必做）

> 每只票必须用一句话提炼核心矛盾，格式：`XX vs YY`

**输出**：一句话回答"公司处在哪个顺风/逆风周期" + 核心矛盾

**6. 量价配合分析**

| 形态 | 特征 | 含义 | 当前状态 |
|------|------|------|---------|
| 放量上涨 | 成交量放大+股价上涨 | 多头强势，资金流入 | ✅/❌ |
| 缩量上涨 | 成交量萎缩+股价上涨 | 惜售，但需警惕量能不足 | ✅/❌ |
| 放量下跌 | 成交量放大+股价下跌 | 恐慌盘涌出，杀跌 | ✅/❌ |
| 缩量下跌 | 成交量萎缩+股价下跌 | 无量阴跌，需等待企稳 | ✅/❌ |
| 量价背离 | 价涨量缩或价跌量增 | 警惕信号，趋势可能反转 | ✅/❌ |

**7. 关键形态识别**

- **突破形态**：突破前高/颈线位，放量确认 → 看涨
- **整理形态**：箱体震荡/三角收敛，等待方向选择 → 观望
- **反转形态**：头肩顶/底、双顶/底，需量价配合确认 → 看跌/看涨

## 六、五维分析体系（补充）

### 9.1 五维分析框架

**权重分配：**
- 基本面（40%）：产业逻辑、业绩弹性、财务质量
- 资金面（20%）：筹码分布、主力资金、机构持仓
- 技术面（15%）：位置、趋势、量价配合
- 情绪面（15%）：市场热度、拥挤度、预期差
- 事件催化（10%）：政策、订单、产能投放

### 9.2 基本面分析（40%）

**核心要素：**
1. **产业逻辑（15%）**
   - 趋势三要素：供需格局、价格锚点、海外验证
   - 产业链位置：是否处于利润集中环节
   - 政策支持：是否有明确政策驱动

2. **业绩弹性（15%）**
   - 价格敏感度：涨价/降本对利润的影响
   - 量的弹性：产能利用率、新产能投放
   - 业绩确定性：订单可见度、客户稳定性

3. **财务质量（10%）**
   - 盈利能力：毛利率、净利率、ROE趋势
   - 现金流：经营性现金流/净利润比值
   - 资产负债：负债率、商誉、应收账款

**评分标准：**
- 优秀（32-40分）：产业趋势确认+业绩高增长+财务健康
- 良好（24-31分）：产业逻辑成立+业绩稳定增长
- 一般（16-23分）：产业逻辑待验证或业绩波动
- 较差（<16分）：产业逻辑不成立或财务恶化

### 9.3 资金面分析（20%）

**核心要素：**
1. **筹码分布（8%）**
   - 股东户数变化：集中度提升=看好
   - 机构持仓占比：>20%=机构认可
   - 北向资金持股：持续增持=外资看好

2. **主力资金（7%）**
   - 近20日主力净流入：持续流入=看好
   - 大单成交占比：>30%=主力活跃
   - 资金流向与股价背离：警惕信号

3. **融资融券（5%）**
   - 融资余额变化：增加=看多情绪
   - 融券余额：异常增加=做空力量
   - 融资买入占比：>10%=杠杆资金活跃

**评分标准：**
- 优秀（16-20分）：筹码集中+主力持续流入+融资活跃
- 良好（12-15分）：筹码稳定+主力净流入
- 一般（8-11分）：筹码分散或资金流向不明
- 较差（<8分）：筹码分散+主力流出+融券增加

### 9.4 技术面分析（15%）

**核心要素：**
1. **位置判断（6%）**
   - 相对历史高低点位置：低位>中位>高位
   - PE/PB历史分位：<30%=低估，>70%=高估
   - 均线系统：多头排列>震荡>空头排列

2. **趋势判断（5%）**
   - 趋势方向：上升趋势>震荡>下降趋势
   - 趋势强度：放量突破>缩量上涨>无量上涨
   - 关键位置：是否突破/跌破重要支撑阻力

3. **量价配合（4%）**
   - 放量上涨：健康
   - 缩量上涨：需警惕量能不足
   - 放量下跌：恐慌盘
   - 量价背离：警惕信号

**评分标准：**
- 优秀（12-15分）：低位+上升趋势+量价配合
- 良好（9-11分）：中位+趋势向上
- 一般（6-8分）：高位或趋势不明
- 较差（<6分）：高位+下降趋势+量价背离

### 9.5 情绪面分析（15%）

**核心要素：**
1. **市场热度（6%）**
   - 板块涨幅排名：前20%=热门
   - 换手率：>5%=活跃，>10%=过热
   - 涨停/跌停家数：板块情绪指标

2. **拥挤度（5%）**
   - 机构持仓占比：>30%=拥挤
   - 近期涨幅：月涨幅>30%需警惕
   - 市场关注度：研报数量、媒体报道

3. **预期差（4%）**
   - 当前处于预期传导哪个阶段
   - 市场已定价多少预期
   - 是否存在超预期空间

**评分标准：**
- 优秀（12-15分）：适度热度+低拥挤度+预期差大
- 良好（9-11分）：热度适中+拥挤度可控
- 一般（6-8分）：热度高+拥挤度上升
- 较差（<6分）：过热+高拥挤度+预期充分

### 9.6 事件催化（10%）

**核心要素：**
1. **政策催化（4%）**
   - 是否有明确政策支持
   - 政策落地时间窗口
   - 政策力度与持续性

2. **业绩催化（3%）**
   - 财报披露时间窗口
   - 业绩预告/快报超预期
   - 订单/合同公告

3. **事件催化（3%）**
   - 产能投放/新产品发布
   - 股权激励/股东增持
   - 行业会议/重大事件

**评分标准：**
- 优秀（8-10分）：多重催化+时间窗口明确
- 良好（6-7分）：有催化+时间窗口可预期
- 一般（4-5分）：催化不明确或时间窗口远
- 较差（<4分）：无催化或负面事件

### 9.7 五维综合评分

**总分计算：**
总分 = 基本面得分×40% + 资金面得分×20% + 技术面得分×15% + 情绪面得分×15% + 事件催化得分×10%

**评级标准：**
- A级（80-100分）：五维共振，强烈推荐
- B级（60-79分）：多维度支持，推荐关注
- C级（40-59分）：部分维度支持，谨慎参与
- D级（<40分）：多维度不支持，不建议参与

## 七、快速筛选与炒作风险识别

### 10.1 19项快速筛选清单

**使用场景：**
- 快速过滤明显不符合框架的标的
- 识别"伪题材"和高炒作风险标的
- 在深度分析前进行初步筛选

**筛选清单（任一项不通过需谨慎）：**

#### 第一层：基本面硬指标（5项）
1. [ ] **市值≥60亿**？（50亿以下信息不透明）
2. [ ] **主营业务清晰**？（不是多元化控股平台）
3. [ ] **最近一年扣非净利润为正**？（排除持续亏损）
4. [ ] **资产负债率<80%**？（排除高杠杆风险）
5. [ ] **无重大诉讼/处罚**？（近1年内）

#### 第二层：产业逻辑验证（5项）
6. [ ] **产业趋势可验证**？（有A级信源支撑）
7. [ ] **趋势三要素≥2/3**？（供需+价格+海外）
8. [ ] **3个季度内能落地业绩**？（不是远期概念）
9. [ ] **公司处于产业链关键环节**？（不是边缘参与者）
10. [ ] **主营业务与题材强相关**？（营收占比>30%）

#### 第三层：估值与位置（4项）
11. [ ] **PE/PB不在历史极高位**？（分位<90%）
12. [ ] **近3个月涨幅<100%**？（排除短期暴涨）
13. [ ] **盈亏比>1:1**？（向上空间>向下空间）
14. [ ] **有明确止损位**？（可量化的逻辑破坏条件）

#### 第四层：资金与交易结构（3项）
15. [ ] **换手率适中**？（日均3-8%，排除过冷/过热）
16. [ ] **机构持仓占比适中**？（10-40%，排除过度拥挤）
17. [ ] **主力资金近20日净流入**？（或至少不持续流出）

#### 第五层：风险排查（2项）
18. [ ] **无"不碰清单"特征**？（蹭概念/贸易商/主业拖累）
19. [ ] **无明显炒作痕迹**？（见下文炒作风险识别）

**通过标准：**
- 19项全通过：可进入深度分析
- 16-18项通过：需重点关注未通过项
- <16项通过：不建议深度分析

### 10.2 炒作风险识别（5个维度）

**维度1：基本面脱节度（20分）**

| 指标 | 低风险（16-20分） | 中风险（11-15分） | 高风险（≤10分） |
|------|-----------------|-----------------|----------------|
| 主营相关度 | 营收占比>50% | 营收占比20-50% | 营收占比<20%或无实际业务 |
| 业绩兑现时间 | 1-2季度内可验证 | 3-4季度内可验证 | >1年或无法验证 |
| 产业地位 | 细分龙头，市占率前三 | 行业参与者 | 边缘参与或蹭概念 |
| 订单/产能支撑 | 有明确订单或产能 | 有规划但未落地 | 无订单无产能 |

**维度2：估值泡沫度（20分）**

| 指标 | 低风险（16-20分） | 中风险（11-15分） | 高风险（≤10分） |
|------|-----------------|-----------------|----------------|
| PE历史分位 | <50% | 50-80% | >80%或亏损 |
| 近3月涨幅 | <30% | 30-80% | >80% |
| 与同业估值比 | 溢价<30% | 溢价30-80% | 溢价>80% |
| PEG | <1 | 1-2 | >2或无法计算 |

**维度3：资金投机度（20分）**

| 指标 | 低风险（16-20分） | 中风险（11-15分） | 高风险（≤10分） |
|------|-----------------|-----------------|----------------|
| 换手率 | 日均3-8% | 日均8-15% | 日均>15% |
| 龙虎榜频率 | 近1月0-1次 | 近1月2-3次 | 近1月>3次 |
| 游资占比 | 龙虎榜游资<30% | 龙虎榜游资30-60% | 龙虎榜游资>60% |
| 融资买入占比 | <5% | 5-10% | >10% |

**维度4：舆情炒作度（20分）**

| 指标 | 低风险（16-20分） | 中风险（11-15分） | 高风险（≤10分） |
|------|-----------------|-----------------|----------------|
| 股吧/社区热度 | 讨论适中，理性分析 | 热度高，情绪化增加 | 刷屏级热度，极度情绪化 |
| 自媒体推荐 | 少量专业分析 | 多个自媒体推荐 | 大量自媒体集中推荐 |
| 概念标签数量 | 1-2个核心概念 | 3-4个概念 | >5个概念（万金油） |
| 公司互动易回复 | 谨慎客观 | 模糊暗示 | 主动蹭热点 |

**维度5：技术形态风险（20分）**

| 指标 | 低风险（16-20分） | 中风险（11-15分） | 高风险（≤10分） |
|------|-----------------|-----------------|----------------|
| 涨停板数量 | 近1月0-2个 | 近1月3-5个 | 近1月>5个 |
| 量价配合 | 放量上涨，健康 | 缩量上涨或量价背离 | 天量滞涨或巨量跌停 |
| 技术位置 | 低位或中位 | 中高位 | 历史高位或加速赶顶 |
| K线形态 | 稳步上涨 | 快速拉升 | 连续一字板或T字板 |

**炒作风险综合评分：**
- 80-100分：低炒作风险，可正常分析
- 60-79分：中等炒作风险，需警惕情绪退潮
- 40-59分：高炒作风险，不建议追高
- <40分：极高炒作风险，远离

### 10.3 "真成长" vs "伪题材" 识别

**真成长特征：**
1. ✅ 产业逻辑有A级信源支撑（政策文件/行业数据/龙头财报）
2. ✅ 公司有实际订单/产能/技术储备
3. ✅ 业绩弹性可量化（价格敏感度公式）
4. ✅ 1-2个季度内可通过财报验证
5. ✅ 估值虽高但有业绩增长支撑（PEG<2）
6. ✅ 机构持仓为主，游资占比低

**伪题材特征：**
1. ❌ 产业逻辑仅有C级信源（自媒体/论坛传闻）
2. ❌ 公司仅有"规划"/"研发中"，无实际业务
3. ❌ 业绩弹性无法量化或逻辑链条断裂
4. ❌ 业绩兑现时间>1年或根本无法兑现
5. ❌ 估值极高且无业绩支撑（PE>100或亏损）
6. ❌ 游资主导，龙虎榜频繁出现

**典型案例对比：**

| 维度 | 真成长案例 | 伪题材案例 |
|------|----------|----------|
| 产业逻辑 | AI算力需求爆发（英伟达财报+国内政策） | 元宇宙概念（仅有媒体炒作） |
| 公司参与度 | 液冷服务器订单落地，产能扩张 | 仅在互动易表示"有关注" |
| 业绩验证 | 下季度财报可验证收入增长 | 无明确业绩兑现路径 |
| 估值 | PE 40倍，PEG 0.8 | PE 150倍或亏损 |
| 资金结构 | 机构持仓40%，北向增持 | 游资龙虎榜占70% |
| 股价表现 | 稳步上涨，回调有支撑 | 连续涨停后闪崩 |

### 10.4 快速筛选工作流

**Step 1：初筛（5分钟）**
1. 查看市值、主营业务、最近财报
2. 检查是否有重大诉讼/处罚
3. 通过→进入Step 2；不通过→放弃

**Step 2：产业逻辑验证（10分钟）**
1. 搜索产业趋势的A级信源
2. 验证公司是否有实际业务参与
3. 评估业绩兑现时间窗口
4. 通过→进入Step 3；不通过→标注"仅题材"

**Step 3：炒作风险评估（5分钟）**
1. 查看近3月涨幅、换手率、龙虎榜
2. 检查股吧/社区热度
3. 计算炒作风险评分
4. 低/中风险→进入深度分析；高风险→放弃或等待回调

**Step 4：深度分析（按Step 0-8执行）**
- 完整执行本框架的Step 0-8流程
- 输出完整研究报告

## 八、完整分析工作流（整合版）

### 11.1 标准工作流（首次覆盖）

**阶段1：快速筛选（20分钟）**
1. 执行19项快速筛选清单
2. 炒作风险识别（5个维度）
3. 判断：真成长 vs 伪题材
4. 决策：进入深度分析 / 放弃 / 观察

**阶段2：基础研究（1-2小时）**
1. Step 0：任务锁定
2. Step 0.5：核心异常分析（v3.1 必做，读 anomalies.md）
3. Step 1：宏观与周期定位
4. Step 2：产业链深度拆解
5. Step 3：公司筛选与质量评分
6. 决策：继续深入 / 降级为"仅观察"

**阶段3：深度分析（2-3小时）**
1. Step 4：业绩弹性测算
2. Step 5：风险分析与逻辑破坏条件
3. Step 6：估值-赔率与买卖时机
   - 资金面分析
   - 技术面分析
   - 期货关联分析（如适用）
   - 研报估值验证
   - 三档目标价体系
4. Step 7：对标与对比分析
5. Step 8：跟踪计划与综合结论

**阶段4：五维综合评分（30分钟）**
1. 基本面评分（40%）
2. 资金面评分（20%）
3. 技术面评分（15%）
4. 情绪面评分（15%）
5. 事件催化评分（10%）
6. 计算总分，确定评级

**阶段5：输出与复核（30分钟）**
1. 撰写完整研究报告（Markdown格式）
2. 执行质量自检清单
3. 生成JavaScript可视化文件
4. 附参考信息源清单

### 11.2 跟踪更新工作流

**高频跟踪（周度，15分钟）**
1. 更新产品现货价格（如适用）
2. 查看资金流向变化
3. 关注行业动态/公司公告
4. 判断：逻辑是否变化？

**季度跟踪（财报发布后，1小时）**
1. 更新财务数据（营收/净利/毛利率/现金流）
2. 验证业绩弹性测算是否准确
3. 更新三档目标价
4. 调整质量评分和五维评分

**事件驱动跟踪（事件发生后，30分钟）**
1. 验证事件真实性（信源级别）
2. 评估对核心逻辑的影响
3. 更新逻辑破坏条件
4. 决策：继续持有 / 加仓 / 减仓 / 离场

### 11.3 决策树

```
标的筛选
├── 19项快速筛选
│   ├── <16项通过 → 放弃
│   └── ≥16项通过 → 炒作风险识别
│       ├── 高/极高风险 → 放弃或等待

│           │       ├── 五维综合评分
│           │       │   ├── A级（80-100） → 强烈推荐

    └── 事件驱动 → 决策：持有/加仓/减仓/离场
```

## 九、案例示范（简化版）

### 案例：某AI算力设备公司

**快速筛选（通过）：**
- ✅ 市值120亿
- ✅ 主营：AI服务器液冷散热设备
- ✅ 2025年扣非净利润3.2亿（+85%）
- ✅ 资产负债率45%
- ✅ 无重大诉讼

**炒作风险识别：75分（中等风险）**
- 基本面脱节度：18分（订单落地，产能扩张）
- 估值泡沫度：12分（PE 45倍，历史70%分位）
- 资金投机度：14分（日均换手8%，龙虎榜2次/月）
- 舆情炒作度：16分（热度适中）
- 技术形态风险：15分（近1月涨幅60%）

**核心矛盾：**
AI算力需求爆发 vs 液冷渗透率提升速度

**趋势三要素：3/3**
- ✅ 供需格局：AI服务器出货量高增长，液冷渗透率从5%→20%
- ✅ 价格锚点：单台液冷设备价值量10-15万，价格稳定
- ✅ 海外验证：英伟达GB200要求液冷，海外数据中心加速改造

**业绩弹性：**
每新增1000台液冷设备 → 年增收入1.2亿 → 年增净利润0.36亿

**三档目标价：**
- 短期（3个月）：35元（2025年PE 40倍）
- 中期（12个月）：50元（2026年PE 35倍）
- 长期（3年）：80元（2028年PE 30倍）

**五维评分：82分（A级）**
- 基本面：35/40（产业逻辑硬+业绩高增长）
- 资金面：16/20（机构持仓35%，北向增持）
- 技术面：11/15（中高位，趋势向上）
- 情绪面：12/15（热度适中，拥挤度可控）
- 事件催化：8/10（英伟达GB200放量+国内政策）

**综合结论：**
- 判断：当前更像"趋势"（产业逻辑硬+业绩兑现中）
- 风险等级：中（估值偏高，需警惕情绪退潮）
- 风格标签：交易型（景气成长）
- 操作建议：择时参与，回调至30元以下可考虑配置

## 附录：产业链拆解实战案例

### C.1 新能源车产业链利润分配演变

```
2020-2021（产能紧缺期）：
上游原材料（碳酸锂/六氟磷酸锂）毛利率70-80% → 利润集中在资源端
中游电池厂（宁德时代）毛利率25-30% → 被上下游挤压
下游整车厂（比亚迪/特斯拉）毛利率15-20% → 涨价传导受限

2023-2025（产能过剩期）：
上游原材料毛利率暴跌至10-20% → 利润向中下游迁移
中游电池厂毛利率回升至30-35% → 规模效应+技术溢价
下游整车厂毛利率分化：
  - 高端品牌（理想/问界）毛利率20-25% → 品牌溢价
  - 低端品牌（零跑/哪吒）毛利率<10% → 价格战受损

投资启示：
- 产能紧缺期买上游资源（赚供需错配的钱）
- 产能过剩期买中游龙头（赚规模效应+技术壁垒的钱）
- 全周期避开低端整车厂（两头受挤，无定价权）
```

### C.2 AI算力产业链利润分配

```
完整传导链：
英伟达GPU（毛利率70%+）
  ↓ 成本占比：服务器总成本的40-50%
AI服务器整机（浪潮/超微，毛利率15-20%）
  ↓ 成本占比：各零部件成本占比
├─ 电源模组（麦格米特，毛利率35%，单台价值>10万）
├─ 高速连接器（中航光电，毛利率40%，国产化率<15%）
├─ PCB主板（沪电股份，毛利率25%，价值量提升5-10倍）
├─ 液冷散热（英维克，毛利率30%+）
└─ MLCC（风华高科，毛利率20%，单台用量万颗级）

利润分配格局：
- 利润最集中：英伟达（垄断地位，70%+毛利率）
- 利润次集中：核心零部件（电源/连接器/液冷，30-40%毛利率）
- 利润最薄：整机组装（浪潮/超微，15-20%毛利率）

投资优先级排序：
1. 高壁垒零部件（电源/连接器/液冷）：毛利率高+供给紧+下游对价格不敏感
2. 英伟达产业链受益者（PCB/覆铜板）：确定性强+量价齐升
3. 整机厂：规模大但利润薄，适合指数配置而非个股深挖
```
