> 本模块产出：估值方法选择 + 三档目标价 + 研报对比 + 盈亏比量化

## 方法论

### Step 6：估值-赔率与买卖时机（主体部分）

> **inputs:** `quote`, `kline_daily`, `blocks.research`, `blocks.recommend`
> **outputs:** 估值方法 + 研报验证 + 三档目标价 [T:] 区间 + 盈亏比
> **注：** 资金面（6a+）与技术面（6a++）拆到 flow-tech 模块

#### 6a. 估值方法选择

| 公司类型 | 适用方法 | 示例 |
|---------|---------|------|
| 成长型 | PE / PEG / PS | 科技股 |
| 周期型 | PB / EV-EBITDA / 单位成本曲线 | 资源股 |
| 困境反转 | PB + 减值出清 + 现金流拐点 | 重组股 |

#### 6b. 期货关联分析（涉及大宗商品的公司必做）

**适用范围：**
- 有色金属（铜、铝、锌、镍、锂、稀土）
- 化工品（纯碱、硫磺、PVC、甲醇、PTA）
- 农产品（玉米、大豆、棉花）
- 能源（煤炭、石油、天然气）

**期货分析四要素：**

| 要素 | 分析内容 | 投资含义 |
|------|---------|---------|
| 基差结构 | 现货价 - 期货价 = 基差（正/负） | 正基差(Back)=现货紧张，负基差(Contango)=预期供给宽松 |
| 持仓结构 | 多空持仓比、主力席位变化 | 产业资本增持=看好后市，投机资金主导=波动加大 |
| 库存周期 | 交易所仓单 + 社会库存 | 库存处于历史低位 = 涨价持续性强 |
| 移仓规律 | 近月-远月价差变化 | 近强远弱=短期供需紧张，远强近弱=预期改善 |

**期货信号与股价传导表：**

| 期货信号 | 传导逻辑 | 股价反应时滞 | 案例 |
|---------|---------|------------|------|
| 期货价格突破前高 | 产业趋势确认 → 业绩预期上修 | 领先 1-2 周 | 碳酸锂期货上涨→赣锋锂业提前启动 |
| 基差持续扩大 | 现货紧张 → 涨价传导至公司 | 同步或滞后 1 周 | 硫磺基差走阔→粤桂股份调价 |
| 持仓量异常放大 | 产业资本对冲/投机炒作 | 需结合基本面判断 | 持仓量暴增但价格不涨=警惕见顶 |
| 库存拐点出现 | 去库→涨价，累库→跌价 | 领先 2-4 周 | LME 铜库存见底→紫金矿业启动 |

**跟踪数据源：**
- 国内期货：上期所/大商所/郑商所/广期所官网
- 海外期货：LME（有色）、COMEX（贵金属）、NYMEX（能源）
- 基差数据：百川盈孚、卓创资讯、我的钢铁网

**目标价合理性检验（必做三项）：**
1. **历史估值分位检验**：当前 PE 处于 __ 分位，目标价对应 PE 处于 __ 分位

3. **业绩兑现路径检验**：目标价隐含增速 __%，需要 __ 业务增长 __%

#### 6f. 框架原则逐条对照（必做）

| 原则 | 对本票的应用 |
|------|------------|
| 低位谈逻辑 / 高位讲情绪 | 当前处于低位/高位？应聚焦逻辑还是情绪？ |
| 宁可错过不可做错 | 是否属于"做错"风险高的标的？ |
| 盈亏比思维 | 向下有支撑吗？向上空间取决于什么？ |
| 逻辑变了吗？ | 核心逻辑是否仍然成立？变的是什么？ |

#### 6g. 买卖时机判断

| 动作 | 条件 |
|------|------|
| 入场 | 产业逻辑确认 + 位置有安全边际 + 催化事件临近 |
| 加仓 | 基本面与盘面共振（财报确认 + 板块启动） |
| 持有 | 反复自问"产业逻辑变了吗？" |
| 离场 | 涨幅已兑现逻辑 / 突发事件破坏逻辑 |

#### 6i. 风格标签确认

| 风格 | 判定条件 | 典型标的 |
|------|---------|---------|
| 配置型 | 龙头 + 稳健增长 + 估值合理，适合核心仓位 | 行业龙头 |

## summary 必填字段（valuation/<ymd>/data.json）

LLM 在写 report.md **之前**，必须把以下字段写回 `<ymd>/data.json` 的 `summary` 子键：

```json
{
  "summary": {
    "targets": {
      "short": 230.0,
      "mid": 290.0,
      "long": 350.0,
      "mid_change_pct": 15.9,
      "base_date": "2026-05-15"
    },
    "rr": 2.8
  }
}
```

**约束**：

- `targets.short/mid/long` 单位：元 / 股，浮点
- `targets.mid_change_pct` = (mid - 现价) / 现价 × 100，正负皆可
- `targets.base_date` = 估值基准日（通常 = data_as_of）
- `rr` = 盈亏比 = (向上空间 ÷ 向下空间)，至少保留 1 位小数
- 与 6d 表格、hero meta-row 严格一致，差异由 verify_consistency.py 兜底
