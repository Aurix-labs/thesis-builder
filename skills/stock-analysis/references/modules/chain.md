# chain · 产业链与趋势 · 方法论

> 本模块产出：标的所处产业链位置与趋势三要素判断
> 数据输入：`<stock>/chain/<ymd>/data.json`（`meta`/`quote`/`business_segments`/`news`）
> 输出文件：同目录 `report.md`，必须以 `<!-- THESIS_SNAPSHOT_START -->` 段开头

## 强约束

- 数字必须带 `[F:]/[C:]/[I:]/[T:]/[GAP:]` 标签（详见 [../tag-spec.md](../tag-spec.md)）
- 最低行数：50 行
- 必含 section：标的速写 / Step 1 宏观 / Step 2 产业链 ASCII 树 / 趋势三要素打勾

## 方法论

### Step 1：宏观与周期定位

> **inputs:** `meta.industry`, `blocks.news`
> **outputs:** 经济周期定位 + 政策环境 + 核心矛盾一句话

#### 1a. 经济周期映射
- 经济阶段：`复苏前期` / `复苏中期` / `复苏后期` / `震荡`
- 公司所处周期与宏观的映射关系
- 政策阶段：`准备期` / `发力期` / `落地分化期`

**经济周期与板块对应：**

| 周期阶段 | 优势板块 | 逻辑 |
|---------|---------|------|
| 复苏前期 | 资源涨价（有色/稀土/化工） | 上游供给刚性，价格先行 |
| 复苏中期 | 设备/制造放量 | PMI回暖→机械设备订单 |
| 复苏后期 | 消费跟随 | 居民收入改善滞后传导 |

#### 1b. 政策与环境扫描
- 核心政策驱动（具体到政策名称、文件编号）
- 外部环境：汇率、地缘、海外需求、监管变化

#### 1c. 核心矛盾提炼（必做）

> 每只票必须用一句话提炼核心矛盾，格式：`XX vs YY`

**输出**：一句话回答"公司处在哪个顺风/逆风周期" + 核心矛盾

### Step 2：产业链深度拆解与趋势验证

> **inputs:** `blocks.business`, `business_segments`, `blocks.news`
> **outputs:** 产业链 ASCII 图 + 趋势三要素打分 + 价值链利润分布表

#### 2a. 题材来源判断（必做，二选一）

| 方法 | 适用场景 | 示例 |
|------|---------|------|
| 消息面筛选法 | 冷门但逻辑硬的涨价/供给收缩方向 | 硫磺涨价 |
| 炒作脉络延伸法 | 主业确定性强，向新方向延伸 | 混动缸体→低空经济→机器人 |

筛选标准：**能否给具体公司带来真实业绩？** 不能则标注为"仅题材"。

#### 2b. 产业链图（必做，Mermaid flowchart）

LLM 必须在 report.md 的 `### 2b. 产业链图` 段落里放一个 ```` ```mermaid ```` fenced block，由 render_html.py 透传渲染。

**模板：**

```mermaid
flowchart LR
  U["上游<br/>关键原材料·包材"]
  M["本公司<br/>核心业务·毛利率 XX%"]:::core
  D["下游<br/>主要应用场景·终端"]
  U --> M --> D
  classDef core stroke:#5e6ad2,stroke-width:1.5px
```

**要求：**
- 节点 ≤ 5 个（保持卡片可读性）
- 必有 `M["..."]:::core`（本公司节点带 lavender 强调）
- 多业务公司：可在 M 节点拆出 M1/M2，仍用 `:::core` 标记
- 文字必须含关键数据（毛利率 / 营收占比），不允许只写标签

如需在 ASCII 形式做"深度树"（业务细节），放在 `### 2b-detail. 产业链 ASCII 树` 子段。`### 2b.` 段必须以 mermaid 块结尾，便于切段。

## summary 必填字段（chain/<ymd>/data.json）

LLM 在写 report.md **之前**，必须把以下字段写回 `<ymd>/data.json` 的 `summary` 子键：

```json
{
  "summary": {
    "industry_phase": "复苏前期 | 复苏中 | 复苏后期 | 震荡",
    "thesis_one_liner": "一句话核心矛盾（< 30 字，HTML hero lead 用）"
  }
}
```

要求：
- `industry_phase` 必须是上述四个枚举之一
- `thesis_one_liner` 简洁，HTML hero 区域直接渲染，避免标签污染（不带 `[I:]` / `[T:]`）

写法（伪代码）：

```python
data = json.loads(data_json_path.read_text())
data["summary"] = {
    "industry_phase": "复苏中",
    "thesis_one_liner": "新能源整车规模化 + 电池外供放量的双轮模型",
}
data_json_path.write_text(json.dumps(data, ensure_ascii=False, indent=2))
```
