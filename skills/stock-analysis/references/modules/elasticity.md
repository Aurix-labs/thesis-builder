## 方法论

### Step 4：业绩弹性测算

> **inputs:** `financials[]`, `business_segments`, `quote.pe`
> **outputs:** 分业务弹性树 + 价格敏感度公式 + 悲观/基准/乐观情景

#### 4a. 分业务弹性树（必做，ASCII 格式）

```
公司：XXX（代码）
核心资产：XXX

├── 业务A：XXX（营收占比XX%）
│   ├── 产能/销量：
│   ├── 单价/毛利率：
│   ├── 价格敏感度：每涨X元/吨 → 年增毛利Y亿
│   └── 弹性来源：
│
├── 业务B：XXX（营收占比XX%）
│   └── ...
│
├── 当前估值
│   ├── 股价/市值/PE/PB：
│   └── 机构一致预期：
│
└── 结论：一句话
```

#### 4b. 价格敏感度公式（必做）

格式：`关键变量每变动 X 单位 → 年利润影响 Y 亿`

弹性公式应满足：
- 可量化（不能只写"明显提升"）
- 与业务结构匹配（如周期股按吨×价差、成长股按出货量×ASP）
- 必须有产能/利用率/毛利率/税率/股本等支撑变量

#### 4c. 情景分析（必做）

| 情景 | 触发条件 | 营收区间 | 净利区间 | 对应 PE |
|------|---------|---------|---------|--------|
| 悲观 |  |  |  |  |
| 基准 |  |  |  |  |
| 乐观 |  |  |  |  |

三档之间要保持内部一致：
- **悲观**：价格回落到周期低位 / 产能利用率显著下行 / 客户流失
- **基准**：当前价格+产能扩张如期 / 一致预期中位
- **乐观**：价格突破前高 + 新业务放量 + 海外订单兑现

每档需注明：净利对应 EPS、PE 区间、市值区间，与 valuation 模块的三档目标价对齐。
