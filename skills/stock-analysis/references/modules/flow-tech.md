## 方法论

### Step 6a+：资金面分析（必做）

**筹码分布分析：**

| 指标 | 数据 | 判断 |
|------|------|------|
| 前 10 大股东持股比例 | XX% | 集中度高/低 |
| 机构持仓占比 | XX% | 机构认可度 |
| 北向资金持股 | XX 亿（占流通 XX%） | 外资态度 |
| 股东户数变化 | 较上期 +/-XX% | 筹码集中/分散 |
| 融资融券余额 | 融资 XX 亿 / 融券 XX 万股 | 做多/做空力量 |

**资金流向分析：**

| 时间周期 | 主力净流入 | 散户净流入 | 判断 |
|---------|-----------|-----------|------|
| 近 5 日 | XX 亿 | XX 亿 | 主力/散户主导 |
| 近 20 日 | XX 亿 | XX 亿 |  |
| 近 60 日 | XX 亿 | XX 亿 |  |

**关键判断：**
- 筹码是否集中？（股东户数下降 + 机构增持 = 筹码集中）
- 主力资金态度？（持续净流入 = 看好，持续流出 = 看空）
- 是否拥挤？（北向 + 公募 + 私募持仓占比 > 30% 需警惕）

### Step 6a++：技术面分析

**数据来源：** Phase 1 输出的 `data.json` 中 `kline_daily` 字段（OHLC + 成交量，最近 3 年约 730 行）。
**分析周期：** 最近 120 个交易日（约 6 个月）。
**计算方式：** 由 agent 在分析时直接对 `kline_daily` 数组做均线 / 布林带 / 历史分位 / 涨跌幅 计算，无需额外脚本。

**1. 价格位置分析**

| 维度 | 数据 | 判断 |
|------|------|------|
| 当前价格 | XX 元 | - |
| 52 周高点 | XX 元（YYYY-MM-DD） | 距离高点 -XX%（低位/中位/高位） |
| 52 周低点 | XX 元（YYYY-MM-DD） | 距离低点 +XX% |
| 历史分位（近 3 年） | XX% | <30% 低估 / 30-70% 合理 / >70% 高估 |
| 布林带位置 | 上轨 XX / 中轨 XX / 下轨 XX | 当前在 XX 位置 |

**2. 均线系统分析**

| 均线 | 价格 | 乖离率 | 排列 | 趋势判断 |
|------|------|--------|------|---------|
| MA5 | XX 元 | +/-XX% | 多头/空头 | 短期趋势 |
| MA10 | XX 元 | +/-XX% | 多头/空头 | 短期趋势 |
| MA20 | XX 元 | +/-XX% | 多头/空头 | 中期趋势 |
| MA60 | XX 元 | +/-XX% | 多头/空头 | 长期趋势 |
| MA120 | XX 元 | +/-XX% | 多头/空头 | 长期趋势 |

**均线排列判断：**
- 多头排列：MA5 > MA10 > MA20 > MA60（看涨）
- 空头排列：MA5 < MA10 < MA20 < MA60（看跌）
- 粘合震荡：均线相互缠绕（方向不明）

**3. 成交量分析**

| 指标 | 数据 | 历史分位 | 判断 |
|------|------|---------|------|
| 今日成交量 | XX 万手 | XX% | 放量/缩量/正常 |
| 今日成交额 | XX 亿元 | XX% | 活跃度 |
| 量比 | X.XX | - | >2 放量 / >1.5 温和放量 / <0.8 缩量 |
| 换手率（日） | XX% | - | >5% 活跃 / >10% 过热 |
| 换手率（周） | XX% | - | 周换手率 |
| 5 日均量 | XX 万手 | - | 对比基准 |

**4. 技术指标**

| 指标 | 数值 | 信号 | 判断 |
|------|------|------|------|
| MACD | DIF: XX, DEA: XX, 柱: XX | 金叉/死叉/多头/空头 | 动能强/弱 |
| RSI(14) | XX | 超买(>70)/超卖(<30)/中性 | 强弱判断 |
| KDJ | K: XX, D: XX, J: XX | 超买(>80)/超卖(<20)/中性 | 短期超买超卖 |
| 布林带宽 | XX% | 收窄/正常/扩张 | 波动率判断 |

**5. 关键价格位（基于历史数据计算）**

**支撑位：**
1. 第一支撑：XX 元（近期低点 / 前期平台 / MA60）
2. 第二支撑：XX 元（前期低点 / 整数关口）
3. 第三支撑：XX 元（历史重要支撑）

**阻力位：**
1. 第一阻力：XX 元（近期高点 / 前期套牢区）
2. 第二阻力：XX 元（前期高点 / 整数关口）
3. 第三阻力：XX 元（历史重要阻力）

**计算方法：**
- 局部极值法：使用 scipy.signal.argrelextrema 识别局部高低点
- 成交密集区：统计历史成交量分布，识别密集成交区
- 整数关口：10 元、20 元、30 元等心理价位

**6. 量价配合分析**

| 形态 | 特征 | 含义 | 当前状态 |
|------|------|------|---------|
| 放量上涨 | 成交量放大 + 股价上涨 | 多头强势，资金流入 | ✅/❌ |
| 缩量上涨 | 成交量萎缩 + 股价上涨 | 惜售，但需警惕量能不足 | ✅/❌ |
| 放量下跌 | 成交量放大 + 股价下跌 | 恐慌盘涌出，杀跌 | ✅/❌ |
| 缩量下跌 | 成交量萎缩 + 股价下跌 | 无量阴跌，需等待企稳 | ✅/❌ |
| 量价背离 | 价涨量缩或价跌量增 | 警惕信号，趋势可能反转 | ✅/❌ |

**7. 关键形态识别**

- **突破形态**：突破前高/颈线位，放量确认 → 看涨
- **整理形态**：箱体震荡/三角收敛，等待方向选择 → 观望
- **反转形态**：头肩顶/底、双顶/底，需量价配合确认 → 看跌/看涨

**8. K 线图可视化（HTML 内嵌）**

HTML 报告中将自动生成：
- 最近 120 日 K 线图（红涨绿跌，中国特色）
- MA5/MA10/MA20/MA60 均线叠加
- 成交量柱状图（红绿配色）
- 支撑位 / 阻力位标注（虚线）
- 可缩放、可拖动、可悬停查看详情

| 趋势 | 5 分 | X/5 | 上升趋势(4-5 分) / 震荡(2-3 分) / 下降(0-1 分) |
| 量价配合 | 4 分 | X/4 | 健康(3-4 分) / 一般(2 分) / 背离(0-1 分) |
| **总分** | **15 分** | **X/15** | 优秀(12-15) / 良好(9-11) / 一般(6-8) / 较差(<6) |
